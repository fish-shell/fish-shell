\section fish_mode_prompt fish_mode_prompt - define the appearance of the mode indicator

\subsection fish_mode_prompt-synopsis Synopsis

The fish_mode_prompt function will output the mode indicator for use in vi-mode.

\subsection fish_mode_prompt-description Description

`fish_mode_prompt`, by default, will output indicators to be displayed in the
mode indicator position to the left of the regular prompt. Declare your own
function to customize the appearance of the mode indicator. You can also
declare an empty fish_mode_prompt function to remove the vi-mode indicators.
The `$fish_bind_mode variable` can be used to determine the current mode. It
will be one of `default`, `insert`, `replace_one`, or `visual`.

\subsection block-example Example

\fish
function fish_mode_prompt
  switch $fish_bind_mode
    case default
      set_color --bold red
      echo 'N'
    case insert
      set_color --bold green
      echo 'I'
    case replace_one
      set_color --bold green
      echo 'R'
    case visual
      set_color --bold brmagenta
      echo 'V'
    case '*'
      set_color --bold red
      echo '?'
  end
  set_color normal
end
\endfish

Outputting multiple lines is not supported in `fish_mode_prompt`.
