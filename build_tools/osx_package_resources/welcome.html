ghoti is a smart and user-friendly command line shell. For more information, visit ghotishell.com.

ghoti will be installed into /usr/local/, and its path will be added to /etc/shells if necessary.

Your default shell will not be changed. To make ghoti your login shell after the installation, run:

chsh -s /usr/local/bin/ghoti

Enjoy! Bugs can be reported on GitHub.
