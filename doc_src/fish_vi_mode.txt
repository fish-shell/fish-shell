\section fish_vi_mode fish_vi_mode - Enable vi mode

\subsection fish_vi_mode-synopsis Synopsis
\fish{synopsis}
fish_vi_mode
\endfish

\subsection fish_vi_mode-description Description

`fish_vi_mode` enters a vi-like command editing mode. To always start in vi mode, edit `$HOME/.config/fish/functions/fish_user_key_bindings.fish` to resemble:

\fish
function fish_user_key_bindings
  fish_vi_key_bindings
end
\endfish
